# GTM-CC Setup Guide

> **Full guide available as PDF:** `docs/gtm-cc-setup-guide.pdf`

## Quick Start (5 Steps)

1. **Install Cursor** — download from cursor.com
2. **Clone the repo** — `git clone https://github.com/cleverviral-mayank/GTM-CC.git`
3. **Open the folder** in Cursor: File → Open Folder → GTM-CC
4. **Download `.env`** from Google Drive (link from Mayank) → drop it into the GTM-CC folder
5. **Type `/whoami`** in the Cursor AI chat → pick your role → start working

## Roles

| # | Role | Access |
|---|------|--------|
| 1 | Copy Strategist | Full access — recipes, clients, segments, all commands |
| 2 | Clay Operator | Pull leads, push to Clay, enrichment, export |
| 3 | Campaign Operator | Check outputs, export CSV (read-only) |

## Commands

**Clay Operator:** `/pull-leads`, `/push-to-clay`, `/generate-http-query`, `/check-outputs`, `/export-csv`

**Campaign Operator:** `/check-outputs`, `/export-csv`

**Copy Strategist:** All of the above + `/create-recipe`, `/test-recipe`, custom SQL

## Keeping Updated

Run `git pull` in the terminal to get the latest commands and rules.

## Need Help?

Screenshot any error and send to the Copy Strategist.
